# curso-web

##  comandos git
``git version`` _ver version_.  
``git status``  _ver el estatus de los archivos._  
``git log`` _ver los commits de la rama._  
``git add {archivo}`` _para introducir a lo que sera confirmado._  
``git add .``  _para introducir todo a lo que sera confirmado._  
``git restore`` _para descartar los cambios en el directorio de trabajo._  
``git push``  _publicar tus commits._  
``git commit -m "nota"`` _poner una etiqueta._   
``git stash`` _apartar cambios._  
``git stash pop`` _regresar los cambios._  
``git checkout -b`` _{nombre de la rama}_  

## terminal
``cd`` moverse entre carpetas  
``ls`` mostrar archivos  
``ls -la`` ordenar archivos  
``clear`` borrar todo  

## vim
``:q`` cerrar y no guardar.  
``i`` insertar.  
``:w`` guardar.
